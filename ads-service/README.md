# SavdoApp Ads Service

Этот сервис предназначен для удаленного управления прямой рекламой в приложении SavdoApp и обеспечения безопасности ключей ИИ.

## Функции

- **Remote Config**: Управление баннерами без обновления приложения.
- **Admin Panel**: Веб-интерфейс для добавления и аналитики рекламных кампаний.
- **AI Proxy**: Защита ключа Groq API через серверный прокси.
- **Analytics**: Сбор статистики кликов по баннерам.

## Развертывание на Vercel

1. Создайте новый проект на Vercel и подключите этот репозиторий.
2. Установите `ads-service` как **Root Directory** в настройках Vercel.
3. Добавьте следующие переменные окружения (Environment Variables):
   - `MONGODB_URI`: Строка подключения к вашему кластеру MongoDB Atlas.
   - `MONGODB_DB_NAME`: Имя базы данных (например, `savdo`).
   - `ADMIN_PASSWORD`: Пароль для входа в `/admin`.
   - `GROQ_API_KEY`: Ваш ключ API от Groq для работы голосового ввода.
4. Нажмите **Deploy**.

## Настройка приложения (Mobile App)

В файле `.env` вашего мобильного приложения установите:

```
EXPO_PUBLIC_ADS_API_URL=https://your-project.vercel.app
```

После этого приложение начнет автоматически загружать рекламу с сервера и использовать прокси для голосового ввода.

## Инициализация базы данных

Вы можете запустить скрипт для создания необходимых коллекций и тестового объявления:

```bash
cd ads-service
npm install
# Создайте .env.local с MONGODB_URI
npm run init-db
```
